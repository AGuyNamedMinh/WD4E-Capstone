### Web Design for Everybody Capstone project

This is the repository of my capstone project website for the [Web Design for Everybody: Basics of Web Development \& Coding Specialization course](https://www.coursera.org/programs/fptu-summer-2026-4c291/specializations/web-design) offered by FPT University on Coursera.
<br>
The website is my portfolio, including the WD4E assignments that I've made.
<br>
It took **a lot of effort** to make this website. Also, implementing a proper layout for the header was a massive fucking headache. It took several attempts of editing both the HTML and CSS in order to make it work properly.
<br>
Oh, and one more thing, the way I designed the website is **not** mobile-first. I know it's a bad practice, I forgot to plan for the mobile layout. Whoops!
